# Commercial Lease Cost — 2026 Metro Dataset

Open dataset of commercial lease cost benchmarks across 25 US metros for Q1 2026:
asking PSF (Class A/B office, retail, industrial), vacancy rates, free-rent
medians, TI allowance ranges. Sources: CBRE / Cushman & Wakefield / JLL / Newmark
quarterly Marketbeat reports + CompStak deal data + LoopNet listings.

## Files
- `metros.json` — 2 metros, structured JSON
- `metros.csv` — same data, CSV format
- `LICENSE` — CC BY 4.0

## Live calculator
https://commercialleasecost.com — free total cost of occupancy calculator using
this same dataset. No email gate, no broker upsell.
